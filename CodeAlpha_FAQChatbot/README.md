# 🤖 FAQ Chatbot

## 📌 Overview

The FAQ Chatbot is a Python desktop application that answers frequently asked customer questions using Natural Language Processing (NLP). It compares the user's question with predefined FAQs using TF-IDF vectorization and cosine similarity to provide the most relevant answer.

---

## 🚀 Features

- User-friendly graphical interface
- Answers Frequently Asked Questions
- Uses Cosine Similarity for question matching
- Fast and accurate responses
- Clear Chat button
- Easy to use

---

## 🛠 Technologies Used

- Python
- Tkinter
- NLTK
- Scikit-learn

## ▶️ How to Run

1. Install Python.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python main.py
```
