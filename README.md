# 🤖 CodeAlpha Artificial Intelligence Internship

Welcome to my **CodeAlpha Artificial Intelligence Internship** repository. This repository contains the projects I completed during my internship, showcasing practical applications of Artificial Intelligence, Natural Language Processing, and Computer Vision using Python.

## 📌 Internship Details

* **Organization:** CodeAlpha
* **Domain:** Artificial Intelligence
* **Language:** Python

## 📂 Completed Projects

### 🌍 Language Translation Tool

A desktop application that translates text between multiple languages through an easy-to-use graphical interface.

### 🤖 FAQ Chatbot

An intelligent chatbot that answers frequently asked questions by matching user queries with predefined FAQs using Natural Language Processing techniques.

### 👁️ Object Detection and Tracking

A real-time object detection application that identifies and labels objects from a live webcam feed using the YOLOv8 model.
