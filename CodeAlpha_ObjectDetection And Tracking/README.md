# 👁️ Object Detection and Tracking

## 📌 Overview

This project performs real-time object detection using the YOLOv8 model and OpenCV. It captures live video from a webcam, detects multiple objects, and displays them with labels and bounding boxes.

---

## 🚀 Features

- Real-time webcam detection
- Detects multiple objects simultaneously
- Uses the YOLOv8 pretrained model
- Displays object labels with bounding boxes
- Fast and accurate detection

---

## 🛠 Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8

## ▶️ How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

Press **Q** to exit.
