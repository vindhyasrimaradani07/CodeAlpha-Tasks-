# 🌍 AI Language Translation Tool

## 📌 Overview

The AI Language Translation Tool is a Python desktop application built using Tkinter and the deep-translator library. It allows users to translate text between multiple languages through a simple and user-friendly graphical interface.

---

## 🚀 Features

- Translate text between multiple languages
- Easy-to-use graphical interface
- Copy translated text to clipboard
- Swap source and target languages
- Clear input and output text
- Supports multiple international languages

---

## 🛠 Technologies Used

- Python
- Tkinter
- deep-translator
- pyperclip

## ▶️ How to Run

1. Install Python.
2. Install the required packages:

pip install -r requirements.txt

3. Run the project:

python main.py
